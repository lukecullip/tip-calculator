# Tip Calculator
An Android application designed to automatically calculate the tip and total cost of a bill!
The Tip Calculator was created using Kotlin, Android Studio, and Jetpack Compose.
Created by Luke Cullip at Towson University under the supervision of Professor Randy Valis.

To run: 
1. Download files locally
2. Open in Android Studio
3. Sync Project with Gradle files
5. Build files
6. Run using plugged in physical Android device or emulator (I use Medium Phone API 36.1)
